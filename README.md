# 📊 Customer Churn Prediction System

An Explainable AI powered telecom customer churn prediction dashboard built using Machine Learning, SHAP Explainability, and Streamlit.

---

# 🚀 Features

- Customer churn prediction
- Random Forest Machine Learning model
- SHAP Explainability
- Risk level analysis
- Business recommendations
- Bulk CSV prediction
- Download prediction reports
- Interactive Streamlit dashboard

---

# 🧠 Machine Learning Models

- Logistic Regression
- Random Forest Classifier

---

# 📈 Explainable AI

This project uses SHAP (SHapley Additive Explanations) to explain why customers are likely to churn.

---

# 🛠️ Tech Stack

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- SHAP
